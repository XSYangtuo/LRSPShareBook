# Visual Studio Code（VSCode）

vscode是一款轻量级但功能强大的**编辑器**。简单来说，它就是一款“瑞士军刀”，一个高级的文本编辑器。
#### Getting Started
- 可以从官网下载安装：[https://code.visualstudio.com/](https://code.visualstudio.com/){:target="_blank"}
- 如果你需要
!!! warning
    vscode和vs不是同一个东西！不要把vscode简称为vs！

## 有什么特点？
1. **轻量快速**：相比完整的IDE，VSCode启动迅速，占用资源少
2. **智能提示**：内置IntelliSense功能，提供代码补全、参数提示、类型信息等
3. **强大调试**：内置调试器，支持断点、变量监视、调用堆栈等调试功能
4. **扩展丰富**：拥有庞大的扩展市场，可以安装各种语言支持、主题、工具等
5. **集成终端**：内置终端，支持PowerShell、CMD、bash等，无需切换窗口
6. **Git集成**：内置Git支持，可以直接进行版本控制操作
7. **跨平台**：在Windows、macOS、Linux上提供一致的体验

## 如何Getting Started？
1. **下载安装**：访问[VSCode官网](https://code.visualstudio.com/)下载对应版本并安装
2. **基本界面**：熟悉资源管理器、编辑器、侧边栏、状态栏等界面元素
3. **安装扩展**：
   - 打开扩展视图（Ctrl+Shift+X）
   - 搜索需要的扩展（如Python、JavaScript、C++等语言支持）
   - 点击安装
4. **常用快捷键**：
   - 打开命令面板：Ctrl+Shift+P
   - 打开文件：Ctrl+P
   - 保存文件：Ctrl+S
   - 格式化文档：Shift+Alt+F
5. **创建第一个项目**：
   - 新建文件夹作为项目目录
   - 用VSCode打开该文件夹
   - 创建新文件并开始编码
