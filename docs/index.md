# 首页
## 这是什么？
本站是一份面向创建、运营、维护**学习资源共享平台（下称之平台）**的同学的教程、文档、经验总结。旨在：
- 帮助有意向搭建平台的同学快速选择合适的搭建方案，并学习基本的开发知识，完成平台搭建。
- 为正在运营平台的团队提供可选择的内容管理规范、资源获取方法、代际传承等方案。
- 提供部分已有平台的经验教训供参考
## 这里有什么？
- [技术选型](techRoute/techRouteIndex.md)：技术选型，选择适合的搭建方案。
- [技术工具](techs/techsIndex.md): 技术工具，介绍一些常用的技术工具。
- [运营Q＆A](topics/topicsIndex.md)：搭建平台后，平台的运营工作也是一个大问题。
- 平台经验：部分已有平台负责人的成功经验和教训。
