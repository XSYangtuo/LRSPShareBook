# 如何保证平台稳定的输入输出资源？
平台说到底就是个资源中转站，没有稳定的资源进进出出，平台就是个空架子。经过我们的调查，在这个问题上，我们需要综合考虑这些因素。

### 稳定资源流转做法的优点
不用愁没东西更。有固定的人负责更新，用户随时来都能看到新东西。
资源质量有保证。有统一的审核标准，不会乱七八糟的东西都往平台上放。
用户体验好。资源不会随便失效，用户想找什么基本都能找到。

### 稳定资源流转做法的缺点
容易依赖少数人。如果几个核心贡献者走了，平台可能直接就断更了。
审核很累人。资源多了之后，每天光审核就要花好几个小时。
资源容易被和谐。不管是网盘链接还是直链，都可能说没就没，维护起来很麻烦。
