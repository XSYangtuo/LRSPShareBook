# 我们要选择校内网还是校外网？
平台开设在校内网还是校外网都是有利有弊的，简单来说，开在校内网意味着只能在校内访问或vpn访问，校外网意味着任何人都可以访问。经过我们的调查，在这个问题上，我们需要综合考虑这些因素。

### 校内网的优点
- 对资源和人的保护。一些隐私性要求较高的资源，以及存在版权争议的资源和存在争议的社会问题，通过校内平台，可以很好地进行保护。
- 认同感。对于一些同学而言，校内网的资源看起来比校外网更容易信任。
### 校内网的缺点
- 访问限制。限制了校外访问，存在大量的访问限制。首先是地点限制。在校外访问资源的同学，需要使用vpn，并不那么容易。其次是校内网本身并不是很稳定。例如ZJU GIT有时候会出现访问失败的情况，而且整个校园网也可能会偶尔爆炸。校内网的访问速度也让人不敢恭维。
- 资源传播度。相比校外网的平台，校内网的平台天然就不好传播。想象你是用户，你拿到一个链接，在校外打不开，开的还很慢...
