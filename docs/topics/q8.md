# 平台传承问题怎么解决
学生团队最大的问题就是铁打的平台流水的兵，每年都有人毕业走。经过我们的调查，在这个问题上，我们需要综合考虑这些因素。

### 做好平台传承的关键做法
- 写好详细的运营手册。把所有操作流程、注意事项都写下来，照着做就行。
- 提前培养接班人。不要等到快毕业了才找人，提前半年就开始带新人。
- 给新人足够的空间和尊重。让他们有参与感和归属感，才愿意留下来。
- 每年搞一次新老成员交流会，把平台的历史和初心传下去。

### 做好传承的优点
- 平台不会断档。老成员走了，新成员能顺利接手，平台能一直运营下去。
- 能少走弯路。把之前踩过的坑、积累的经验都传下去，后来的人不用再重新摸索。
- 能保持平台特色。不会因为换了一批人，平台就变得面目全非。

### 平台传承的缺点
- 经验很难完全传递。很多东西是只可意会不可言传的，老成员走了就带走了。
- 交接很麻烦。需要花很多时间带新人，教他们怎么运营平台。
- 新人可能没动力。平台不是自己一手建起来的，可能没有那么强的责任心。
